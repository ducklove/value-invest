알림 채널

포트폴리오 조건(지정가·목표가 달성·등락률·총평가액)을 만족하면 메신저로 알려드립니다. 받을 채널을 본인 계정으로 연결한 뒤, 아래 ‘알림 규칙’에서 조건을 추가하세요. 연결·규칙은 모두 본인에게만 적용됩니다.

① 텔레그램 — 내 봇 토큰으로 등록
  1. 텔레그램에서 @BotFather를 열고 /newbot으로 봇을 만든 뒤 봇 토큰(예: 123456:ABC-...)을 복사합니다.
  2. 아래 ‘텔레그램’ 칸에 토큰을 붙여넣고 [연결]을 누릅니다.
  3. 방금 만든 봇과 대화를 열어 아무 메시지나 보낸 뒤 다시 [연결]을 누르면 chat_id가 자동 감지됩니다.
    원하면 chat_id를 직접 입력해도 됩니다(@userinfobot로 확인). 텔레그램 계정이 없으면 먼저 가입하세요.
  4. 연결되면 [테스트]로 실제 수신을 확인하세요.
② 카카오톡 — 내 카카오 앱으로 등록
  1. developers.kakao.com에서 앱을 만들고 REST API 키를 복사합니다. 본인 앱이면 본인이 관리자라 ‘나에게 보내기’가 검수 없이 됩니다.
  2. 그 앱에서 카카오 로그인 활성화, 동의항목 ‘카카오톡 메시지 전송’ 사용 설정, Redirect URI에 아래 카카오 칸에 표시되는 주소를 등록합니다.
  3. 아래 ‘카카오톡’ 칸에 REST 키를 붙여넣고 [카카오 연결] → 카카오 로그인·동의.
  4. 연결되면 [테스트]로 확인하세요.

· 텔레그램과 카카오를 모두 연결하면 양쪽으로 동시에 받습니다. · 각 줄의 버튼으로 임시 끄기·해제를 언제든 할 수 있습니다. · 입력한 토큰/키는 본인 계정에만 저장되어 본인에게 보내는 데만 쓰입니다.
